# 🌾 AgriSmart - Digital Krishi Mitra

AgriSmart is a web-based application that helps farmers plan crop activities efficiently using smart recommendations.

## 🚀 Features
- Crop Calendar (month-wise planning)
- Irrigation Tips
- ROI Calculator
- Weather Forecast
- Eco-Friendly Advisory
- Government Schemes Information

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📌 Purpose
To assist farmers in making data-driven decisions and improve productivity.
